# Minesweeper-Game

본 프로그램은 **9×9 격자**에 **10개의 지뢰**를 **무작위**로 배치하고, 사용자가 좌표를 입력해 안전한 칸을 찾아가는 **콘솔 기반 지뢰찾기 게임**입니다.  
게임의 전체 흐름은 **초기화 → 지뢰 배치 → 사용자 입력 → 주변 탐색 → 승패 판정** 단계로 구성됩니다.  

---

### Logic  

💣 난수 생성 함수인 'rand()'를 사용해 지뢰를 무작위로 배치하며,  
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**인접한 8방향을 검사하는 탐색 알고리즘**(8-directional search)을 통해 선택한 칸 주변의 지뢰 개수를 계산합니다.  

🔍 게임 상태는 2차원 배열 `play[][]` 와 `check[][]` 로 관리되며, 두 배열을 이용해 지뢰의 위치와 플레이어의 선택 상태를 함께 관리합니다.
