# Minesweeper-Game  

본 프로젝트는 **C 언어로 구현한 콘솔 기반 지뢰찾기 게임**으로, **9×9 격자 내에 10개의 지뢰를 무작위로 배치**하고 사용자가 좌표를 입력해 안전한 칸을 찾아가는 구조로 설계되었습니다.  
게임의 전체 흐름은 **초기화 → 지뢰 배치 → 사용자 입력 → 지뢰 여부 판정 → 인접 지뢰 수 계산 → 승패 판정** 단계로 이루어져 있습니다.  

---

### ⚙️ 주요 특징
- **`rand()` 함수**를 사용하여 지뢰의 위치를 **무작위(random)** 로 배치  
- **8방향 탐색 알고리즘(8-directional search)** 을 통해 선택한 칸 주변의 지뢰 개수를 계산  
- **2차원 배열(`play[][]`, `check[][]`)** 을 이용해 **지뢰의 실제 위치**와 **사용자의 선택 상태**를 관리  
- **`winning_checker()` 함수**를 통해 **승패 판정 알고리즘** 구현  

---

### 📘 한계 및 배운 점  
본 프로젝트를 통해 **2차원 배열**, **반복문**, **함수 호출 간의 상호 작용**을 활용하여 하나의 완전한 게임 로직을 직접 설계·구현하는 경험을 쌓을 수 있었습니다.
